Play Queue

Play Queueとは？

Youtube動画を、連続再生するプレイヤーです。

tl;dr ... 3タブ構成です。左[再生タブ]の関連動画と、右[検索タブ]の検索動画から、中央[プレイリスト]に動画を追加しましょう！自動で連続再生されます。

タブ構成

1. play_arrow再生

動画の再生/停止、関連動画をさがすことができます。
「↓」ボタンを押すと関連動画をさらに表示します。
再生したい動画をクリックして、プレイリストに追加しましょう。 「今すぐ再生」「次に再生」「最後に再生」の3つで、追加されるプレイリスト内の場所が異なります。

2. playlist_playプレイリスト

プレイリストを編集、保存できます。再生順を並べ替えたり、動画を削除したりすることが可能です。
プレイリストは上から下に順に再生されていきます。
(プレイリストに新しい動画を追加するときは他タブから行います。関連動画/検索動画から探し追加してください。)

2.1 プレイリスト終了後の動作

下のトグルボタンではプレイリストの末尾の動画が再生終了したときの動作を選べます。

  • ループ:
    ONにすると、プレイリスト一番上の動画を再生します。
  • 末尾の関連動画を自動再生する:
    ONにすると、プレイリスト末尾の関連動画の中で一番上のものを再生します。 (「ループ」がOFFのときのみ選べます。)
  • 新しい関連動画のみ:
    ONにすると、「末尾の関連動画を自動再生する」で再生される動画が既にプレイリストにある時、その動画ではなく次に関連する動画を再生します。 (「末尾の関連動画を自動再生する」がOFFのときのみ選べます。)

2.2 保存/開く

プレイリストを保存したり、保存したプレイリストを読み込むことができます。
2.2.1 現在のプレイリストを保存

最も下にある「現在のプレイリストを保存」ボタンを押すと、現在のプレイリストに名前を付けて保存することができます。 ボタンを押した後、「プレイリスト名」欄にプレイリストの名前を入力してください。 存在しない名前のプレイリストは新規作成され、同じ名前のプレイリストは上書きされます。

2.2.2 過去のプレイリストを開く

「過去のプレイリストを開く」ボタンを押し、次に出てくる「保存したリスト」プルダウンメニューを開くと端末に保存されているPlayQueueのプレイリストが表示されます。 一つ選択し「開く」ボタンを押すと当該プレイリストを読み込みます。
保存されたプレイリストのリネーム、編集はできませんので、一度削除して再び保存してください。

2.2.3 過去のプレイリストを削除

「過去のプレイリストを削除」ボタンを押し、次に出てくる「保存したリスト」プルダウンメニューを開くと端末に保存されているPlayQueueのプレイリストが表示されます。 一つ選択し「削除」ボタンを押すと当該プレイリストを削除します。このとき確認画面が表示されます。復元できませんので注意してください。

3. search検索

上部の検索バーに検索ワードを入力してEnterまたは下の「↓」ボタンを押すと、Youtube動画を検索できます。
再生したい動画をクリックして、プレイリストに追加しましょう。 「今すぐ再生」「次に再生」「最後に再生」の3つで、追加されるプレイリスト内の場所が異なります。
